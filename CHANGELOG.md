# Changelog

All notable changes to this project will be documented in this file.

The format follows Keep a Changelog and Semantic Versioning.

## [1.0.0] - 2026-03-17

- Initialized Next.js 16 + TypeScript + Tailwind v4 portfolio foundation.
- Implemented a professional SaaS-style design system with tactile UI patterns.
- Added dashboard route and projects route with ISR interval (60s).
- Added reusable UI, shared components, and project service scaffolding.
- Added Motion animation support and parallax hero behavior.

## [1.1.0] - 2026-03-17

- Refactored terminology to professional portfolio language across layout and pages.
- Replaced textured game styling with matte dark dashboard visuals (#0B0E14).
- Shifted portfolio content to AI automation and engineering solutions focus.
